## Description
This project is an MVC CRUD application that uses NestJS, Handlebars, MongoDB, and Docker. The purpose was to test how feasible it is to develop an MVC-type application in NestJS.

## Installation
```bash
$ npm install
```

## Running the app

```bash
# development
$ docker compose up -d

# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```
